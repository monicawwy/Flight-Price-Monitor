# 香港機票價格監察器 (HK Flight Price Monitor)

自動搜尋從香港出發嘅平價機票，用 GitHub Actions 定期執行。

## 功能
- 🔍 自動搜尋從香港出發嘅航班
- 💰 追蹤最平機票價格
- 📊 儲存歷史價格數據
- ⏰ 每日自動執行搜尋

## 使用技術
- Python 3.10+
- Amadeus Flight API
- GitHub Actions
- Pandas
